🛠️ TROUBLESHOOTING
====================

npm install fails
  → npm install -g npm@latest
  → npm cache clean --force
  → npm install

Telegram not responding
  → Check .env TELEGRAM_CHAT_ID is correct
  → Open bot and send /help manually
  → Check Node console for errors

Dashboard won't load
  → Verify http://localhost:3000 accessible
  → Check admin password in .env correct
  → Try Ctrl+Shift+Delete (clear cache)

No API data
  → Check internet connection
  → Try: curl https://api.binance.com/api/v3/ticker/price?symbol=BTCUSDT
  → Check Node console logs
